# Cross-run benchmarks

Appended by `pm-scorer` — **one row per completed run**, anonymised (vertical + market, not brand name). Purpose: over time, category scores and proxy metrics get calibrated against what "normal" looks like per vertical instead of judged in a vacuum. Never rewrite or delete existing rows; corrections get a new row with a note.

Column notes: scores are the category 0–100 internals (— when the preset omits the category or the metric was a `data_gap`); `ER%` = best-platform engagement-rate proxy; `ads` = active Meta + Google counts at capture; `cadence` = posts/wk best platform; `LCP` = homepage mobile seconds; `rev/mo` = review velocity per §3.7.
